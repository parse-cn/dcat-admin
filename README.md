## Dcat admin csp friendly

### 安装

```
#composer.json

...
 "require": {
    "parse-cn/laravel-admin": "2.*",
    "spatie/laravel-csp": "^2.10"
},
...
"repositories": [
    {
        "type": "git",
        "url": "git@github.com:parse-cn/dcat-admin.git"
    }
],
"minimum-stability": "dev",
...
```

后面和安装 dcat-admin 一样
